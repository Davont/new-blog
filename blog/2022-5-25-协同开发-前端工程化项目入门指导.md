# 快速开始

## 初始化项目

为了方便快速开始使用，在安装脚手架之前，请确保您已经拥有 `node` 与 `npm` 的开发环境。

## 启动项目

安装依赖

```bash
npm install
```

运行项目
```bash
# Mac OS or Windows
npm run dev

# Linux
npm run dev:linux
```
`npm` 本身是跨平台的，这两条命令的区别在于启动 DevServer 时，是否主动在浏览器打开相应的页面。

tip: 如启动失败，请通过 node -v 检查 Node 版本是否在 12.0.0 以上。
